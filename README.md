# ☁️ AWS High Availability Web Application

## 🚀 Project Overview
A scalable and highly available web application deployed on AWS using EC2 and Load Balancer.

---

## 🏗️ Architecture
User → Application Load Balancer → EC2 Instances

---

## ☁️ AWS Services Used
- EC2
- Application Load Balancer
- Target Group
- AMI
- Security Groups

---

## ⚙️ Implementation Steps
1. Launch EC2 instances
2. Install Apache server
3. Create simple web pages
4. Configure Target Group
5. Setup Load Balancer
6. Test traffic flow
7. Create AMI
